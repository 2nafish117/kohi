Detailed Description

This file contains structures and functions specific to the event system.

Events are a mechenism that allows the developer to send and recieve data at critical points in the execution of the application in a non- coupled way. For now, this follows a simple pub-sub model of event transmission.

typedef struct event_context event_context

Represents event contextual data to be sent along with an event code when an event is fired. It is a union that is 128 bits in size, meaning data can be mixed and matched as required by the developer.
